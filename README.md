# Note App React

### Package in Project:
- React Router Dom
- Framer Motion
- React Toastify

## [See Website](https://noteapp-reactjs.netlify.app/)

##  Run 

Clone the project

```bash

git clone https://github.com/mohammadyousefvand/Note-App-React.git

```

Go to the project directory

```bash

cd Note-App-React

```

Install dependencies

```bash

npm install

```

Start the server

```bash

npm start

```
